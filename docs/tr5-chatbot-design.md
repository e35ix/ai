# تصميم روبوت الدردشة TR5

## نظرة عامة على التصميم

تطبيق TR5 هو روبوت دردشة احترافي يتميز بتصميم داكن أنيق مع تفاعلات سلسة وأنيميشن متقدم. يهدف التصميم إلى توفير تجربة مستخدم استثنائية مع التركيز على الوضوح والأناقة.

## لوحة الألوان

### الألوان الأساسية
- **الخلفية الرئيسية**: `#000000` (أسود نقي)
- **الخلفية الثانوية**: `#171717` (رمادي داكن جداً)
- **النص الأساسي**: `#FFFFFF` (أبيض نقي)
- **النص الثانوي**: `#737373` (رمادي متوسط)
- **النص الفاتح**: `#CCCCCC` (رمادي فاتح)
- **اللون المميز**: `#F1EAE0` (بيج فاتح للتباين)

### الألوان التفاعلية
- **الحالة النشطة**: `#FFFFFF` مع شفافية 10%
- **الحالة المحومة**: `#737373` مع شفافية 20%
- **الحدود**: `#171717` مع شفافية 30%

## هيكل التطبيق

### 1. الهيدر (Header)
- شعار TR5 في الجانب الأيسر
- عنوان "روبوت الدردشة الذكي" في المنتصف
- أيقونة الإعدادات في الجانب الأيمن
- خلفية شفافة مع تأثير blur

### 2. منطقة الدردشة الرئيسية
- خلفية سوداء نقية
- رسائل المستخدم: محاذاة يمين، خلفية `#171717`
- رسائل الروبوت: محاذاة يسار، خلفية `#0a0a0a`
- فواصل زمنية بين الرسائل
- مؤشر الكتابة المتحرك

### 3. منطقة الإدخال (Input Area)
- حقل نص مع placeholder ديناميكي
- زر الإرسال مع أيقونة متحركة
- مؤشر حالة الاتصال
- دعم السحب والإفلات للملفات

### 4. الشريط الجانبي (Sidebar)
- قائمة المحادثات السابقة
- خيارات التخصيص
- معلومات الحساب
- إعدادات النموذج

## التفاعلات والأنيميشن

### 1. أنيميشن الرسائل
```css
/* ظهور الرسائل الجديدة */
@keyframes messageSlideIn {
  from {
    opacity: 0;
    transform: translateY(20px) scale(0.95);
  }
  to {
    opacity: 1;
    transform: translateY(0) scale(1);
  }
}

/* تأثير الكتابة */
@keyframes typing {
  0%, 60%, 100% {
    transform: translateY(0);
  }
  30% {
    transform: translateY(-10px);
  }
}
```

### 2. تأثيرات التفاعل
- **Hover Effects**: تغيير لطيف في الشفافية والحجم
- **Click Animations**: تأثير ripple عند النقر
- **Loading States**: مؤشرات تحميل متحركة
- **Smooth Transitions**: انتقالات سلسة بين الحالات

### 3. أنيميشن مؤشر الكتابة
```css
.typing-indicator {
  display: flex;
  align-items: center;
  gap: 4px;
}

.typing-dot {
  width: 8px;
  height: 8px;
  border-radius: 50%;
  background: #737373;
  animation: typingBounce 1.4s infinite ease-in-out;
}

.typing-dot:nth-child(1) { animation-delay: -0.32s; }
.typing-dot:nth-child(2) { animation-delay: -0.16s; }
.typing-dot:nth-child(3) { animation-delay: 0s; }

@keyframes typingBounce {
  0%, 80%, 100% {
    transform: scale(0);
    opacity: 0.5;
  }
  40% {
    transform: scale(1);
    opacity: 1;
  }
}
```

## مكونات واجهة المستخدم

### 1. بطاقة الرسالة (Message Card)
- حدود مدورة (border-radius: 16px)
- ظلال خفيفة للعمق
- أيقونة المرسل
- طابع زمني
- حالة التسليم

### 2. حقل الإدخال (Input Field)
- تصميم مسطح مع حدود متوهجة عند التركيز
- placeholder متحرك
- عداد الأحرف
- أزرار الإجراءات المدمجة

### 3. أزرار التفاعل
- تصميم مسطح مع تأثيرات hover
- أيقونات SVG متحركة
- ردود فعل لمسية (haptic feedback)
- حالات مختلفة (عادي، محمل، معطل)

## التخطيط المتجاوب (Responsive Layout)

### Desktop (1200px+)
- شريط جانبي ثابت (300px)
- منطقة دردشة رئيسية (مرنة)
- هيدر ثابت في الأعلى

### Tablet (768px - 1199px)
- شريط جانبي قابل للطي
- تخطيط مكدس
- تحسين أحجام الخطوط

### Mobile (< 768px)
- تخطيط عمودي كامل
- شريط جانبي منزلق
- أزرار أكبر للمس
- تحسين المساحات

## الخلاصة

تصميم TR5 يجمع بين الأناقة والوظائف العملية، مع التركيز على تجربة مستخدم سلسة ومتطورة. الألوان الداكنة تخلق بيئة مريحة للعين، بينما الأنيميشن والتفاعلات تضيف طابعاً حيوياً للتطبيق.
